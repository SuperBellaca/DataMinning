Minería de datos

Un dataset es una colección estructurada de datos que sirve como base para análisis o entrenamiento de modelos. Los datasets suelen estar organizados en filas y columnas, donde cada fila representa una instancia o registro, y cada columna, una variable o característica específica del fenómeno que se está estudiando.
